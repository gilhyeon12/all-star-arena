# All Star Arena

파이썬(Python)과 Pygame 라이브러리를 사용하여 제작된 2D 대전 격투 게임 프로젝트입니다. 나루토와 블리치의 캐릭터가 등장하여 대결을 펼칩니다.

## 🎮 게임 특징

이 게임은 고전적인 2D 격투 게임의 재미를 살려 제작되었습니다.

*   **다양한 게임 모드**:
    *   **1인용 모드 (vs Bot)**: AI 봇과 대결하여 실력을 쌓을 수 있습니다.
    *   **2인용 모드 (PvP)**: 하나의 키보드로 친구와 함께 대전할 수 있습니다.
*   **전략적 전투 시스템**:
    *   **체력 & 마나**: 체력 바 하단의 하늘색 게이지는 마나입니다.
    *   **스킬 시스템**: 기본 공격 외에 마나를 소모하는 강력한 스킬 3가지를 사용할 수 있습니다.
    *   **방어 & 대쉬**: 적의 공격을 방어하거나 대쉬로 빠르게 이동하는 전술적인 플레이가 가능합니다.
*   **캐릭터**:
    *   현재 나루토와 이치고 캐릭터를 플레이할 수 있습니다.
    *   `create_pixel_art.py` 스크립트를 통해 픽셀 아트 리소스를 생성하여 사용합니다.

## 🛠️ 설치 및 실행 방법

### 1. 환경 설정
Python 3가 설치되어 있어야 합니다. 터미널에서 아래 명령어를 실행하여 가상 환경을 설정하고 필요한 라이브러리를 설치하세요.

```bash
# 가상 환경 생성 (필요 시)
python3 -m venv myenv

# 가상 환경 진입 (Linux/Mac)
source myenv/bin/activate
# Windows의 경우: myenv\Scripts\activate

# 의존성 설치
pip install pygame
```

### 2. 게임 실행
프로젝트 루트 디렉토리에서 다음 명령어로 게임을 시작합니다.

```bash
python -m game.main
```

## 🕹️ 조작법

| 기능 | Player 1 (왼쪽) | Player 2 (오른쪽/숫자패드) |
| :--- | :--- | :--- |
| **이동** | `A` (좌), `D` (우) | `←` (좌), `→` (우) |
| **점프** | `W` | `↑` |
| **기본 공격** | `J` | `Num 1` |
| **방어 (Guard)** | `K` (데미지 80% 감소) | `Num 2` |
| **대쉬 (Dash)** | `L` (마나 소모) | `Num 3` |
| **스킬 1** | `U` | `Num 4` |
| **스킬 2** | `I` | `Num 5` |
| **스킬 3** | `O` | `Num 6` |
| **캐릭터 선택** | `A/D` 이동, `Space` 선택 | `←/→` 이동 |

> **참고**: Player 2 조작 시 NumLock이 켜져 있는지 확인해주세요.

## 📂 프로젝트 구조

*   `game/`: 게임의 핵심 로직이 담긴 패키지입니다.
    *   `main.py`: 게임의 진입점 및 메인 루프.
    *   `characters.py`: 캐릭터 클래스 및 동작 정의.
    *   `effects.py`: 이펙트 및 시각 효과 처리.
    *   `settings.py`: 게임 상수 및 설정 값.
*   `create_pixel_art.py`: 게임에 필요한 캐릭터 스프라이트 이미지를 생성하는 스크립트입니다.
*   `USAGE.md`: 상세한 사용 설명서입니다.
