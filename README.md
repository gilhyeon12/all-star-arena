# All Star Arena

파이썬(Python)과 Pygame 라이브러리를 사용하여 제작된 2D 대전 격투 게임 프로젝트입니다. 다양한 애니메이션 캐릭터와 게임 고유 캐릭터가 등장하여 대결을 펼칩니다.

## 🎮 게임 특징

이 게임은 고전적인 2D 격투 게임의 재미를 살려 제작되었습니다.

*   **다양한 게임 모드**:
    *   **1인용 모드 (vs Bot)**: AI 봇과 대결하여 실력을 쌓을 수 있습니다.
    *   **2인용 모드 (PvP)**: 하나의 키보드로 친구와 함께 대전할 수 있습니다.
    *   **라운드 선택**: 단판 또는 3판 2선 모드 선택 가능
*   **전략적 전투 시스템**:
    *   **체력 & 기력**: 체력 바 하단의 오렌지색 게이지는 기력(Ki)이며, 스킬 사용에 필요합니다.
    *   **스킬 시스템**: 기본 공격 외에 기력을 소모하는 강력한 스킬들을 사용할 수 있습니다.
    *   **방어 & 대쉬**: 적의 공격을 방어하거나 대쉬로 빠르게 이동할 수 있습니다. (대쉬는 기력을 소모하지 않습니다.)
*   **풍부한 캐릭터 라인업 (그리드 레이아웃)**:
    *   **나루토, 블리치, 드래곤볼, 원피스, 주술회전** 등 인기 애니메이션 시리즈 캐릭터 등장.
    *   모든 캐릭터가 한눈에 보이는 그리드 UI를 통해 빠르고 직관적인 선택이 가능합니다.
    *   `create_new_characters.py` 스크립트를 통해 대규모 캐릭터 리소스를 일괄 생성합니다.

## 🌟 캐릭터 소개

### 🎭 참전 시리즈 및 캐릭터
이 게임은 모든 캐릭터를 한눈에 볼 수 있는 **그리드 기반 UI**를 지원하며, 선택 시 상단 바에서 시리즈 분류를 확인할 수 있습니다.

*   **Naruto**: Naruto, Sasuke, Sakura
*   **Bleach**: Ichigo, Inoue
*   **Dragon Ball**: Goku, Vegeta
*   **One Piece**: Luffy, Zoro
*   **My Hero Academia**: Todoroki, Uraraka
*   **Attack on Titan**: Eren, Mikasa
*   **Demon Slayer**: Tanjiro, Nezuko
*   **Chainsaw Man**: Denji, Power
*   **Jujutsu Kaisen**: Itadori, Gojo, Fushiguro

### 🤝 보조 캐릭터 (Support)
전투 중 **`O` (P1)** 또는 **`Num 6` (P2)** 키로 소환할 수 있습니다. 
*   소환된 보조 캐릭터는 특수 기술을 사용한 후 사라집니다.
*   사용 후 **20초의 쿨타임**이 존재하며 HUD에서 상태를 확인할 수 있습니다.
*   메인 로스터의 대다수 캐릭터를 보조 캐릭터로 선택하여 전략적으로 활용 가능합니다.

## 🛠️ 설치 및 실행 방법

### 1. 환경 설정
Python 3가 설치되어 있어야 합니다. 터미널에서 아래 명령어를 실행하여 가상 환경을 설정하고 필요한 라이브러리를 설치하세요.

```bash
# 가상 환경 생성 (필요 시)
python3 -m venv myenv

# 가상 환경 진입 (Linux/Mac)
source myenv/bin/activate
# Windows의 경우: myenv\Scripts\activate

# 의존성 설치
pip install pygame
```

### 2. 게임 실행
프로젝트 루트 디렉토리에서 다음 명령어로 게임을 시작합니다.

```bash
python -m game.main
```
또는
```bash
./myenv/bin/python3 -m game.main
```

## 🕹️ 조작법

| 기능 | Player 1 (왼쪽) | Player 2 (오른쪽) |
| :--- | :--- | :--- |
| **이동** | `W` (점프), `S` (방어), `A` (좌), `D` (우) | `↑` (점프), `↓` (방어), `←` (좌), `→` (우) |
| **기본 공격** | `J` (연타 시 콤보) | `Num 1` |
| **점프** | `K` (이중 점프 가능) | `Num 2` (혹은 `↑`) |
| **대쉬 (순보)** | `L` (회피/추격) | `Num 3` |
| **보조 기술** | `U` (원거리/특수기) | `Num 4` |
| **필살기** | `I` (기 1줄 소모, W/S+I 가능) | `Num 5` (기 1줄 소모, ↑/↓+5 가능) |
| **서포터 (교대)** | `O` | `Num 6` |
| **각성 (변신)** | `J` + `K` (기 3줄 소모) | `Num 1` + `Num 2` |
| **합동 공격** | `I` + `O` (기 5줄 소모, 보조와 합동 공격) | `Num 5` + `Num 6` (기 5줄 소모) |

### 메뉴 조작 (캐릭터 선택)

| 기능 | Player 1 | Player 2 |
| :--- | :--- | :--- |
| **메뉴 이동** | `W, A, S, D` | 방향키 (`↑, ↓, ←, →`) |
| **선택/확정** | `J` | `Num 1` |

> **팁**: **Settings(설정) -> Key Settings** 메뉴에서 인게임 조작키를 자유롭게 변경할 수 있습니다. 
> Player 2가 숫자패드를 사용할 경우 NumLock이 켜져 있는지 확인해주세요.

## 📂 프로젝트 구조

    *   `game/`: 게임의 핵심 로직이 담긴 패키지입니다.
    *   `main.py`: 게임의 진입점 및 메인 루프.
    *   `characters.py`: 캐릭터 클래스 및 동작 정의.
    *   `effects.py`: 이펙트 및 시각 효과 처리.
    *   `settings.py`: 게임 상수 및 설정 값.
    *   `support.py`: 보조 캐릭터 시스템.
*   `create_pixel_art.py`: 기존 캐릭터(나루토, 이치고) 스프라이트 이미지를 생성하는 스크립트입니다.
*   `create_new_characters.py`: 새로운 캐릭터 이미지를 생성하는 스크립트입니다.
*   `USAGE.md`: 상세한 사용 설명서입니다.
