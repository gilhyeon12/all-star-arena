# 나루토 vs 블리치 (Python & Pygame)

파이썬과 Pygame 라이브러리로 제작된 2D 격투 게임입니다.

## 실행 방법

1.  **가상 환경 설정 및 설치**:
    터미널에서 아래 명령어를 순서대로 실행하여 가상 환경을 만들고 Pygame을 설치해주세요.
    ```bash
    # 가상 환경 생성 (pip 없이) 및 pip 설치
    python3 -m venv myenv --without-pip
    curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
    ./myenv/bin/python3 get-pip.py
    
    # Pygame 설치
    ./myenv/bin/pip install pygame
    ```

2.  **게임 실행**:
    프로젝트 루트 디렉토리(`pr1`)에서 아래 명령어를 실행하세요.
    ```bash
    ./myenv/bin/python3 -m game.main
    ```

## 게임 모드
1.  **메인 메뉴**:
    -   `1`: **1인용 (vs Bot)** 모드. 봇 AI와 대결합니다.
    -   `2`: **2인용 (PvP)** 모드. 친구와 대결합니다.
2.  **캐릭터 선택**:
    -   Player 1: `A` / `D` 로 선택, `Space`로 확정.
    -   Player 2: `←` / `→` 로 선택 (2인용만).

## 조작법 및 스킬

| 기능 | Player 1 (키보드 왼쪽) | Player 2 (숫자 패드) |
| :--- | :--- | :--- |
| **이동** | `A` (좌), `D` (우) | `←` (좌), `→` (우) |
| **점프** | `W` | `↑` |
| **기본 공격** | `J` | `Num 1` |
| **방어 (Guard)** | `K` | `Num 2` |
| **대쉬 (Dash)** | `L` | `Num 3` |
| **스킬 1** | `U` | `Num 4` |
| **스킬 2** | `I` | `Num 5` |
| **스킬 3** | `O` | `Num 6` |

## 게임 특징 및 시스템
-   **마나 시스템**: 체력 바 아래 **하늘색 게이지**가 마나입니다.
    -   스킬과 대쉬 사용 시 소모되며, 시간이 지나면 자동 회복됩니다.
-   **스킬 전략**:
    -   **기본 공격 (`J`/`1`)**: 마나 소모 없음. 견제 및 콤보용.
    -   **방어 (`K`/`2`)**: 누르고 있으면 데미지를 **80% 감소**시키고 밀려나지 않습니다.
    -   **대쉬 (`L`/`3`)**: 마나를 사용하여 빠르게 접근하거나 도망칩니다.
    -   **스킬 (`U,I,O`/`4,5,6`)**: 강력할수록 더 많은 마나를 소모합니다.
-   **봇 AI**: 1인용 모드에서는 AI가 상황에 맞춰 방어, 대쉬, 스킬을 사용합니다.


## 문제 해결
-   `Num` 키가 작동하지 않으면 키보드의 **NumLock**이 켜져 있는지 확인하세요.


## 문제 해결
-   `ModuleNotFoundError: No module named 'pygame'` 에러가 발생하면 위 1번 단계의 설치가 제대로 되었는지 확인해주세요.
