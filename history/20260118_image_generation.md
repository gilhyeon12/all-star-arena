# 캐릭터 이미지 생성 (AI 대체)

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
AI 이미지 생성 서버 오류로 인해, Python 스크립트(`create_pixel_art.py`)를 작성하여 나루토와 이치고의 특징을 살린 픽셀 아트 이미지를 직접 생성하고 적용했습니다.

### 1. 적용된 이미지
-   **나루토**: 노란 뾰족머리, 파란 서클렛, 주황색 옷.
-   **이치고**: 주황색 머리, 검은 사패장, 등 뒤의 참월(대검).

### 2. 기술적 해결
-   `pygame.draw` 기능을 활용하여 캐릭터의 외형을 픽셀 단위로 그렸습니다.
-   투명 배경(Alpha Channel)을 지원하여 게임 내에서 깔끔하게 보입니다.

## 실행
게임을 다시 실행하면 새로운 캐릭터 이미지가 적용된 것을 확인할 수 있습니다.
```bash
./myenv/bin/python3 -m game.main
```
