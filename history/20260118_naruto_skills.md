# 나루토 스킬 및 이펙트 구현

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
나루토 캐릭터의 고유 스킬 3종과 대쉬 이펙트를 구현했습니다.

### 1. 스킬 목록 (Key: U, I, O)
-   **U (수리검)**: 전방으로 수리검을 던지는 투사체 공격. (MP 5)
-   **I (분신술)**: 체력 15를 가진 분신을 소환하여 함께 싸웁니다. 분신은 Hard 난이도의 AI를 가집니다. (MP 30)
-   **O (나선환)**: 전방으로 빠르게 돌진하며, 부딪히는 적에게 큰 데미지(50)를 입힙니다. (MP 50)

### 2. 시각 효과 (Visual Effects)
-   **대쉬 잔상 (Afterimage)**: 대쉬(`L`) 사용 시 캐릭터 뒤에 반투명한 잔상이 남아 속도감을 줍니다.
-   **투사체**: 수리검 이미지가 전방으로 날아갑니다.
-   **분신**: 본체와 구분하기 위해 약간 투명하게(Alpha 200) 렌더링됩니다.

### 3. 구조 변경
-   `game/objects.py`: 투사체(`Projectile`) 클래스 추가.
-   `game/effects.py`: 잔상(`Afterimage`) 클래스 추가.
-   `game/characters.py`: 스킬 타입(`projectile`, `summon`, `dash_attack`) 처리를 위한 `attack()` 메서드 확장.
-   `game/main.py`: 투사체 및 분신 업데이트, 충돌 처리 로직 추가.

## 실행
```bash
./myenv/bin/python3 -m game.main
```
