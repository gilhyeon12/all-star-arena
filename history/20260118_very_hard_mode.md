# 난이도 확장: VERY HARD 추가

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
사용자의 요청에 따라 `VERY HARD` 난이도를 추가하여 1인용 모드의 도전 요소를 강화했습니다.

### 1. VERY HARD 특징
-   **체력 보너스**: 봇의 체력이 150으로 증가합니다. (기본 100)
-   **공격성 대폭 증가**: 쉴 새 없이 몰아치며 스킬을 난사합니다.
-   **지능적 행동**:
    -   플레이어 공격 시에만 방어를 시도합니다(예측 방어).
    -   위험하거나 기회일 때 **대쉬**를 사용하여 접근/회피합니다.
    -   반응 속도가 매우 빠릅니다.

## 사용 방법
1.  메인 메뉴에서 `3` 또는 `O`를 눌러 설정(Settings)으로 이동합니다.
2.  `4`를 눌러 Difficulty를 **VERY HARD**로 설정합니다.
3.  `B`를 눌러 메뉴로 돌아온 뒤 `1` (1P Mode)로 게임을 시작합니다.
