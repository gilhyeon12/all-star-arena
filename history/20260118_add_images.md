# 캐릭터 이미지 적용

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
단순 색상 사각형으로 표시되던 캐릭터를 이미지 파일(`png`)로 표시하도록 변경했습니다.

### 상세 내용
1.  **디렉토리 생성**: `game/images/` 폴더를 생성했습니다.
2.  **이미지 리소스**:
    -   AI 이미지 생성이 일시적으로 불가능하여, 파이썬 스크립트를 통해 플레이스홀더(Placeholder) 이미지를 자동 생성했습니다.
    -   `naruto.png` (주황색, N 이니셜)
    -   `ichigo.png` (검정색, B 이니셜)
3.  **코드 수정**:
    -   `game/characters.py`: `__init__` 메서드가 색상 대신 이미지 경로를 받도록 수정하고, `pygame.image.load()`를 사용해 이미지를 로드합니다.
    -   `game/main.py`: 캐릭터 생성 시 이미지 파일 경로를 전달합니다.

### 참고
추후 더 멋진 이지(`png`)를 구하시면 `game/images/` 폴더의 파일을 덮어쓰기만 하면 게임에 즉시 반영됩니다.
