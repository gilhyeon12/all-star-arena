# 캐릭터 애니메이션 시스템 구현

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
캐릭터가 상태에 따라 생동감 있게 움직이도록 **애니메이션 시스템**을 구현했습니다.

### 1. 스프라이트 프레임 (Sprite Frames)
-   `create_pixel_art.py`를 업데이트하여 각 동작별로 여러 장의 프레임(`png`)을 생성합니다.
-   **구현된 동작**:
    -   `idle`: 숨쉬기 (2프레임)
    -   `walk`: 걷기 (2프레임)
    -   `run`: 대쉬 (잔상 효과)
    -   `attack`: 공격 모션
    -   `guard`: 방어 자세
    -   `jump`: 점프/체공 자세
    -   `hit`: 피격 자세

### 2. 애니메이션 엔진 (Character Class)
-   `update_state()`: 현재 캐릭터의 행동(이동, 공격, 점프 등)을 분석하여 상태(`self.state`)를 결정합니다.
-   `update_animation()`: 현재 상태에 맞는 이미지를 일정 시간 간격(150ms)으로 교체하여 애니메이션을 재생합니다.

### 3. 결과
-   가만히 있을 때도 숨을 쉬듯 움직입니다.
-   걸을 때 다리가 교차하며 움직입니다.
-   공격, 방어, 점프 시 시각적 피드백이 명확해졌습니다.

## 실행
```bash
./myenv/bin/python3 -m game.main
```
