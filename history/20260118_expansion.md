# 게임 확장: 스킬, 모드, 이펙트 추가

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
사용자의 요청에 따라 게임의 깊이를 더하기 위해 대규모 업데이트를 진행했습니다.

### 1. 게임 구조 개편 (State Machine)
-   `Title` -> `Character Select` -> `Battle` -> `Game Over` 로 이어지는 게임 흐름을 구현했습니다.
-   이제 게임 시작 시 모드(1인용/2인용)를 선택할 수 있습니다.

### 2. 캐릭터 선택 화면
-   Player 1과 Player 2가 각각 나루토 또는 이치고(블리치)를 자유롭게 선택할 수 있습니다.

### 3. 스킬 시스템 도입
단순 공격을 넘어 각 캐릭터별 6가지 스킬 키를 매핑했습니다.
-   **Player 1**: `U`, `I`, `O` (상단), `J`, `K`, `L` (하단)
-   **Player 2**: `Num 4`, `5`, `6` (상단), `Num 1`, `2`, `3` (하단)
-   각 스킬은 쿨타임과 고유의 범위를 가집니다.

### 4. 비주얼 이펙트 (Particles)
-   타격 및 스킬 사용 시 화려함을 더해줄 파티클 시스템(`game/effects.py`)을 추가했습니다.
-   노란색, 주황색, 파란색 등 스킬 속성에 맞는 색상 효과가 나옵니다.

### 5. 봇 AI (1인용 모드)
-   친구 없이도 즐길 수 있도록 1인용 모드를 추가했습니다.
-   AI는 거리 조절, 추격, 확률적 스킬 사용 등의 지능을 가집니다.

## 실행 방법
```bash
./myenv/bin/python3 -m game.main
```
(Player 2는 키패드 NumLock을 켜주세요!)
