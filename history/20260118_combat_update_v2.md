# 전투 시스템 대규모 업데이트: 마나, 방어, 대쉬

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
전투의 전략성을 높이기 위해 마나 시스템과 특수 조작(방어, 대쉬)을 추가하고 키 배치를 전면 수정했습니다.

### 1. 조작 키 재배치 및 역할 변경
하단 키 라인(`J,K,L` / `Num 1,2,3`)을 특수 액션 전용으로 변경했습니다.

| 기능 | P1 (키보드) | P2 (키패드) | 설명 |
| :--- | :--- | :--- | :--- |
| **기본 공격** | `J` | `1` | 마나 소모 없음. 빠른 견제. |
| **방어 (Guard)** | `K` | `2` | 누르고 있으면 데미지 80% 감소 & 넉백 무시. |
| **대쉬 (Dash)** | `L` | `3` | 마나를 소모하여 짧은 거리를 빠르게 이동. |
| **스킬 1** | `U` | `4` | 마나 소모(10). 약한 데미지. |
| **스킬 2** | `I` | `5` | 마나 소모(20). 중간 데미지. |
| **스킬 3** | `O` | `6` | 마나 소모(40). 강한 데미지(필살기). |

### 2. 마나 (Mana) 시스템
-   체력 바 아래에 **하늘색 마나 바**가 추가되었습니다.
-   최대 마나는 100이며, 시간이 지나면 서서히 회복됩니다.
-   스킬과 대쉬 사용 시 마나가 소모됩니다. 마나가 부족하면 기술이 나가지 않습니다.

### 3. 밸런싱
-   스킬마다 데미지와 쿨타임, 마나 소모량이 다르게 적용됩니다.
-   기본 공격은 약하지만 쿨타임이 짧아 콤보 연결에 유리합니다.
-   필살기(`O` / `6`)는 강력하지만 마나 소모가 크고 쿨타임이 깁니다.

## 실행

./myenv/bin/python3 -m game.main
