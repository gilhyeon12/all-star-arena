# 게임 프로젝트 초기 생성

**날짜**: 2026-01-18
**작성자**: Antigravity

## 변경 내용
사용자 요청에 따라 "나루토 vs 블리치" 파이썬 게임 프로젝트를 생성했습니다.

### 생성된 파일
- `game/main.py`: 게임 메인 진입점 및 루프 구현
- `game/settings.py`: 화면 크기, 색상, FPS 등 설정 정의
- `game/characters.py`: 캐릭터 클래스 구현 (이동, 점프, 중력)
- `game/utils.py`: 텍스트 그리기 등 유틸리티 함수
- `USAGE.md`: 사용 설명서

### 기술 스택
- **언어**: Python 3
- **라이브러리**: Pygame

### 이슈
- 시스템에 `pip` 및 `python3-venv`가 설치되어 있지 않아 자동 설치에 실패했습니다. 사용자에게 수동 설치 안내가 필요합니다.
