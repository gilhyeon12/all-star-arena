This folder is used to place the multilingual translated text of the game.
The file name should be an operating system language abbreviation string.
Here is a list of all supported languages and abbreviations for Adobe AIR.

Dutch                    nl

Russian                  ru
Chinese(Simplified)      zh-CN
Spanish                  es
Swedish                  sv
Chinese(Traditional)     zh-TW
Turkish                  tr


If you want to create a translation for a specific language, you can visit the following URL to help us improve the translation.
https://crowdin.com/project/bleach-vs-naruto-sports

If your language is not in the supported list, type the name of your language (without the suffix) in the [extra.txt] file.
